# Budget Allocation System

A full-stack web application for democratic budget allocation through team voting. Team members can submit proposals, vote on them, and collectively decide how to distribute a shared budget in a fair and transparent way.

## Features

- 4-phase workflow: **Setup → Proposal → Voting → Summary**
- Role-based access for **Admin** and **Members**
- Proposal creation and submission
- Voting system with a **0–3 scale**
- Real-time budget tracking
- Final results with accepted and rejected proposals
- System restart option for admins

## Tech Stack

**Frontend**
- React 18
- Vite
- React Router DOM
- CSS3

**Backend**
- Express.js
- SQLite
- REST API

## Prerequisites

Make sure you have installed:

- Node.js v16 or higher
- npm v8 or higher
- Git
